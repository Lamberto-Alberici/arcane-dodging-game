# Arcane Dodging Game

Interactive dodging game built with Java featuring character sprites from the TV show Arcane.

## How to Play

1. Clone or download this repository
2. Compile the Java files
3. Run `MyApp.java` to start the game
4. Drag your mouse to control your avatar and dodge incoming bombs!

## Features

- Mouse-controlled gameplay
- Multiple playable Arcane characters (Vi, Jinx, Caitlyn)
- Dynamic difficulty scaling
- Real-time collision detection

## Course Project

Built for Principles of Software Development at DePauw University.

## Note

Character sprites are from the Netflix series Arcane and are used for educational purposes only.
